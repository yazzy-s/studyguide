Unit 1

Linear Independence & Bases — Probs 1–6

Linearly independent? — vectors in R^n
  1. Put the vectors as columns of a matrix
  2. Row reduce to echelon form
  3. Count pivots. Pivots = number of vectors → independent. Any free variable (column without a pivot) → dependent
e.g. Prob 1a: [1,7,7], [2,7,7], [3,7,7]
put as columns → row reduce → rows 2 and 3 zero out → only 1 pivot 3 vectors, 1 pivot → DEPENDENT
Linearly independent? — polynomials in Pn
  1. Pull coefficients from each polynomial into a row vector — e.g. 3 − x + 9x² → [3, −1, 9]
  2. Put those rows into a matrix and row reduce
  3. Same rule: pivots = number of polynomials → independent
e.g. Prob 2a
3−x+9x² → [3, −1, 9] 5−6x+3x² → [5, −6, 3] 1+x−5x² → [1, 1, −5] put as rows → row reduce → count pivots
Is this set a basis?
  1. Check linear independence (above)
  2. Count vectors — must equal dim of the space. dim(R^n) = n, dim(Pn) = n+1, dim(R^(m×n)) = m·n
  3. Both conditions → basis. Either fails → not a basis
  4. Square matrix shortcut: det ≠ 0 → basis, det = 0 → not a basis
e.g. Prob 3b: only 2 vectors for P2 (dim = 3) → cannot be a basis no matter what
Find basis for a subspace — spanning set given (Prob 6)
  1. Put the spanning vectors as rows of a matrix
  2. Row reduce to echelon form
  3. The nonzero rows of the echelon form = basis
e.g. Prob 6
put [1,2,1], [3,1,−1], [1,−3,−3] as rows → row reduce → keep nonzero rows
Find basis for a subspace defined by a constraint (Probs 5, 20)
  1. Use the constraint to solve for one variable. e.g. a2 − 2a1 = a0 → a0 = a2 − 2a1
  2. Write the general element of the space, substituting in
  3. Group by each free variable — each group becomes a basis vector
  4. Count the groups = dimension
e.g. Prob 5a: { a2x² + a1x + a0 : a2 − 2a1 = a0 }
a0 = a2 − 2a1 general element: (a2−2a1) + a1x + a2x² group by a1: a1·(−2 + x) group by a2: a2·(1 + x²) basis: { −2+x, 1+x² } dim = 2
Unit 2

Dot Products & Orthogonality — Probs 7–13

Compute dot product and length
u·v = u1·v1 + u2·v2 + u3·v3 ||v|| = sqrt(v·v)
  1. Multiply entry by entry, add everything up
  2. Result = 0 → orthogonal. Result ≠ 0 → not orthogonal
e.g. Prob 8b: u = [1,1,1], v = [1,2,−3]
u·v = 1(1) + 1(2) + 1(−3) = 0 → orthogonal ✓
Parallel vectors (Prob 9)
|u·v| = ||u|| · ||v||

Parallel means the angle is 0° or 180°, so cos(θ) = ±1. The absolute value of the dot product equals the product of the lengths.

Is v orthogonal to all of span{v1, v2}? (Prob 11b)
  1. Compute v·v1 and v·v2 separately
  2. Both = 0 → orthogonal to the whole span
  3. Either ≠ 0 → not orthogonal to the span

Dot product distributes: v·(c1·v1 + c2·v2) = c1(v·v1) + c2(v·v2). If v kills each basis vector it kills every combo.

Show span{v} = solution set of ax + by = 0 (Probs 10, 11c)
  1. Plug v into the equation — verify it satisfies it
  2. Solve the system generally by row reducing — express in terms of free variables
  3. Show the general solution is exactly scalar multiples of v
Find orthogonal complement S⊥ (Probs 7, 11–12)
  1. Get a basis for S — call the vectors b1, b2, ...
  2. Let w = (x, y, z) be unknown. Write one equation per basis vector: w·b1 = 0, w·b2 = 0, ...
  3. Collect those equations into a homogeneous linear system — put coefficients in a matrix
  4. Row reduce, find free variables, write general solution
  5. Direction vectors from free variables = basis for S⊥
e.g. Prob 7: S has basis { [1,2,1], [0,1,1] }
w·[1,2,1] = 0 → x + 2y + z = 0 w·[0,1,1] = 0 → y + z = 0 matrix [[1,2,1],[0,1,1]] → row reduce → y=−z, x=z w = z·[1,−1,1] → basis for S⊥: { [1,−1,1] }
Check: dim(S) + dim(S⊥) = dim of the whole space
Unit 3

Spaces, Rank & Null Space — Probs 14–16, 20–22, 38–42

Is vector b in Col(A)? (Probs 14, 50)
  1. Form augmented matrix [A | b]
  2. Row reduce the whole thing
  3. Any row like [0 0 ... 0 | nonzero] → inconsistent → NOT in Col(A)
  4. No such row → IS in Col(A)
e.g. Prob 14c: A = [[1,1],[1,1]], b = [1,3]
augmented: [[1,1|1],[1,1|3]] → R2−R1 → [[1,1|1],[0,0|2]] row [0,0|2] → inconsistent → NOT in Col(A)
Find rank of a matrix (Prob 22)
  1. Row reduce to echelon form
  2. Count nonzero rows = number of pivots = rank

rank = dim(row space) = dim(col space). Zero matrix → rank 0. Max possible rank = min(m, n).

Null space of a MAP h — which inputs give output 0? (Probs 38–40)
  1. Plug a general input into h and set output = 0
  2. Solve for what the input must look like — that set is the null space
e.g. Prob 38: h(p(x)) = x·p(x), domain = P3
need x·p(x) = 0 for all x → only p(x) = 0 works null space = {0}, nullity = 0
Range space of a MAP h — what can h actually output? (Probs 38–40)
  1. Plug a general input into h and observe what form the output always takes
  2. Describe that set of all possible outputs — that is the range space
e.g. Prob 38: h(p(x)) = x·p(x), domain = P3
general input: a0 + a1x + a2x² + a3x³ output: a0x + a1x² + a2x³ + a3x⁴ → output ALWAYS has no constant term range = { polynomials in P4 with no constant term }
Is a specific thing in the null space or range space? (Prob 38a–e)
  1. Null space check: plug it into h as the input. Does h(thing) = 0? Yes → in null space. No → not.
  2. Range space check: work backwards. Set h(input) = thing and solve for input. Valid solution in the domain → in range. No valid solution → not in range.
Prob 38a — is x³ in the range of h(p(x)) = x·p(x)?
need x·p(x) = x³ → p(x) = x² x² is in P3 (the domain) ✓ → YES, x³ is in the range
Prob 38c — is 7 in the range?
need x·p(x) = 7 → p(x) = 7/x → not a polynomial → NOT in the range
Prob 38b — is 0 in the null space?
h(0) = x·0 = 0 ✓ → YES, 0 is in the null space
Null space of a MATRIX H — solve Hx = 0 (Probs 39–40)
  1. Row reduce [H | 0]
  2. Identify free variables (columns with no pivot)
  3. Express each pivot variable in terms of the free variables
  4. Write general solution — the direction vectors = basis for the null space
e.g. H = [[1,2],[2,4]]
[[1,2|0],[2,4|0]] R2−2R1 → [[1,2|0],[0,0|0]] x2 is free, x1 = −2x2 → null space = span{ [−2, 1] }
Rank–Nullity (Prob 41)
rank + nullity = dim(domain)
  1. Find dim(domain): R^n → n, Pn → n+1, R^(m×n) → m·n
  2. Know rank → nullity = dim(domain) − rank
  3. Know nullity → rank = dim(domain) − nullity
Prob 41a: h: R^5 → R^8, rank = 5
nullity = 5 − 5 = 0
Prob 41b: h: P3 → P3, rank = 1
dim(P3) = 4, nullity = 4 − 1 = 3
Unit 4

Linear Maps & Matrices — Probs 23–25, 43–50

Is h a linear map? (Probs 23–25)
  1. Check h(0) = 0 first — if not, stop, it's not linear
  2. Check h(u + v) = h(u) + h(v)
  3. Check h(cu) = c·h(u)
Instant fails — no need to check further: output has an added constant (e.g. +1 anywhere), any variable is squared, two different input components are multiplied together, absolute value appears anywhere
e.g. Prob 23c: h(x,y,z) = [1, 1]
h(0,0,0) = [1,1] ≠ [0,0] → NOT linear, stop
e.g. Prob 23d: h(x,y,z) = [2x+y, 3y−4z]
h(0,0,0) = [0,0] ✓ all terms linear → IS a linear map
Find matrix H using standard bases (Probs 45, 48a)
  1. Apply h to each standard basis vector: e1 = [1,0,...], e2 = [0,1,...], etc.
  2. Each result becomes one column of H, placed left to right
  3. The matrix satisfies H·v = h(v) for any v
e.g. Prob 45: h(e1) = [2,2,0], h(e2) = [0,1,−1]
H = [ [2, 0], [2, 1], [0, −1] ] (each h(ei) is a column)
Where does h send a specific vector? (Probs 44, 48b)
  1. Write the vector as a linear combination of vectors whose images you know
  2. Use linearity: h(c1·v1 + c2·v2) = c1·h(v1) + c2·h(v2)
  3. Substitute in the known outputs and simplify
e.g. Prob 44: h(1)=1+x, h(x)=1+2x, h(x²)=x−x³
want h(1 − 3x + 2x²) = 1·h(1) + (−3)·h(x) + 2·h(x²) = (1+x) − 3(1+2x) + 2(x−x³) = 1+x − 3−6x + 2x−2x³ = −2 − 3x − 2x³
Find Rep_B(v) — v's coordinates in basis B
  1. Set up augmented matrix [b1 | b2 | ... | v] with B vectors as columns, v on the right
  2. Row reduce
  3. Solution gives c1, c2, ... such that v = c1·b1 + c2·b2 + ...
  4. Rep_B(v) = [c1, c2, ...]^T
B = { [1,1], [1,0] }, find Rep_B([3,1])
[[1,1|3],[1,0|1]] → row reduce → c1=1, c2=2 Rep_B([3,1]) = [1, 2]^T check: 1·[1,1] + 2·[1,0] = [3,1] ✓
Build H_{B,D} for general bases (Probs 43a, 47, 48c)
  1. For each basis vector b in B: compute h(b), using linearity if needed
  2. Express h(b) in terms of the D basis vectors — set up [d1 | d2 | ... | h(b)] and row reduce to get h(b)'s D-coordinates
  3. Those D-coordinates = one column of H_{B,D}
  4. Repeat for every b in B — stack columns left to right in the same order as B
H_{B,D} · Rep_B(v) = Rep_D(h(v))
e.g. B={b1,b2}, D={d1,d2}, h(b1)=3d1+2d2, h(b2)=d1−d2
col 1 = D-coords of h(b1) = [3, 2]^T col 2 = D-coords of h(b2) = [1, −1]^T H_{B,D} = [ [3, 1], [2, −1] ]
Unit 5

Inverses, Change of Basis & Determinants — Probs 57–75

Inverse of a 2×2 (Prob 62)
[a b; c d]^(−1) = 1/(ad−bc) · [d, −b; −c, a]
  1. Compute ad − bc. If = 0 → no inverse, stop
  2. Swap a and d, negate b and c
  3. Divide the whole matrix by ad − bc
e.g. Prob 62a: [[2, 1], [−1, 1]]
det = 2(1) − 1(−1) = 3 inverse = (1/3) · [[1, −1], [1, 2]]
Inverse of a larger matrix — Gauss-Jordan (Prob 64)
  1. Write augmented matrix [A | I] where I is the identity of the same size as A
  2. Row reduce the left side all the way to I
  3. Whatever the right side has become = A^(−1)
  4. If left side gets a zero row → singular, no inverse
Change of basis: D → Standard E (Prob 67)
  1. Put the D basis vectors as columns to form matrix M
  2. M converts D-coordinates into standard coordinates
  3. M^(−1) converts standard coordinates into D-coordinates
e.g. Prob 67: D = { [2,1], [−2,4] }
M = [[2,−2],[1,4]] (columns = D basis vectors) M: D-coords → standard. M^(−1): standard → D-coords
Change of basis: B → D general (Prob 68)
  1. Put D-vectors as columns on the left, B-vectors as columns on the right
  2. Form [D | B] and row reduce until left side = I
  3. Right side is now the change-of-basis matrix from B to D

Same idea as Gauss-Jordan for an inverse — you're expressing each B vector in D-coordinates.

Determinant via row reduction (Probs 72–73)
  1. Start a running multiplier = 1
  2. Swap two rows → multiply running multiplier by −1
  3. Scale a row by k → multiply running multiplier by k (divide back out at the end!)
  4. Add a multiple of one row to another → multiplier unchanged
  5. Once in echelon form: multiply all diagonal entries together, then multiply by the running multiplier
e.g. swap once, scale a row by 3 for convenience, reach echelon diag [2, −1, 4]
after swap: multiplier = −1 after scale by 3: multiplier = −3 diagonal product: 2·(−1)·4 = −8 raw result: (−3)·(−8) = 24 scaled by 3 just for convenience → divide back out: 24/3 = 8 det = 8
Key trap: scaling a row changes the det. If you scale to make arithmetic easier, you MUST divide it back out at the end.
Left vs. right inverse (Prob 61)
  1. H is m×n with rank r
  2. rank = m → right inverse exists. H is wide (m < n), map is onto
  3. rank = n → left inverse exists. H is tall (m > n), map is one-to-one
  4. m = n and full rank → regular two-sided inverse
